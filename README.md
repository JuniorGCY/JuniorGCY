### Conceitos e Arquitetura
Além das ferramentas, foco em aplicar boas práticas no desenvolvimento de software:

- 📱 **Mobile-First & Responsividade:** Interfaces pensadas primeiro para a melhor experiência em dispositivos móveis.
- 🏗️ **Design Patterns:** Aplicação de padrões de projeto para criar código limpo e escalável.
- 🎨 **UI/UX Design:** Desenvolvimento focado no usuário, utilizando princípios de Design System com Figma.
- ♻️ **Clean Code:** Escrevendo código legível, testável e de fácil manutenção.
